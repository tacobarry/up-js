# First Staps JavaScript Developer

Curso destinado a capacitação de profissionais utilizando tecnologias JavaScript.

Este curso explora desde os fundamentos da linguagem JavaScript como backend até seu uso no frontend consumindo uma API REST.
A proposta é construirmos um sistema simples de controle de gastos pessoais (se der tempo).

Mas o `foco` será no nivelamento do JavaScript básico.

- Introdução;
- Strict Mode;
- Variáveis;
- Tipos de dados;
- Operadores;
- Estruturas de controle;
- Funções;
- Prototype;
- Shadowing;
- Closure;
- Hoisting;
- Namespace;
- Race Condition;
- Promises;
